‚ö° STRIDE Threat Analysis

Memory Aid: "Smart Thieves Really Investigate Data Everywhere"
Spoofing - Tampering - Repudiation - Information Disclosure - Denial of Service - Elevation of Privilege
üë§ S - Spoofing Threats

Impersonating legitimate users or systems

  • Fake Job Seekers: Creating false profiles to access employer data
  • Fake Employers: Scammers posting jobs to collect resumes
  • Account Takeover: Using stolen credentials to impersonate users
  • Recruiter Impersonation: Unauthorized access to candidate data
  • API Spoofing: Fake applications calling our APIs
  • Email Spoofing: Phishing emails appearing from the platform
Impact: Data theft, fraud, platform reputation damage
üîß T - Tampering Threats

Unauthorized modification of data or systems

  • Resume Manipulation: Altering qualifications or experience
  • Job Posting Fraud: Modifying salary ranges or requirements
  • Search Result Manipulation: Artificial ranking boosts
  • Salary Data Corruption: Skewing market compensation data
  • Application Status Changes: Unauthorized hiring decision updates
  • Algorithm Manipulation: Biasing matching and recommendations
